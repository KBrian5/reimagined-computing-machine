This is a better version of the Apple timer design. In this design the user is able to input the number of hours, minutes, and seconds. The user is less likely to input the wrong digits and could easily change the timer if needed. This is a faster way of setting up a timer and takes away all the scrolling to just seup a timer. The user can now just enter a number for the designated time without having to scroll and wait to find the correct hour, minutes, or seconds.
