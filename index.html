Apple Timer Design

The image above is a screenshot of the timer that is installed in an iPhone. This timer does the job, but the design is not very helpful. In order to set a timer, the user must scroll down with the finger to find the correct hour, minute, and second. This design makes the user take a longer time to set up a timer and makes it easier for the user to input a wrong digit.
